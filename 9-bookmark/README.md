# 9 Bookmark

Chrome new-tab extension inspired by the supplied LumiList reference screens, branded as **9 Bookmark**.

## Install locally

1. Open `chrome://extensions` in Chrome.
2. Enable **Developer mode**.
3. Click **Load unpacked** and choose this folder.
4. Open a new tab to use 9 Bookmark.

The extension stores its workspace locally in Chrome. It supports visual Pages and Boards, draggable boards and links, Chrome-bookmark import, Ctrl+Shift+Y quick saves, privacy blur, private-window link opening, settings, wallpaper, bookmark search, and board-level 6-digit locks.

Locked-board domains are redirected away in normal Chrome tabs. To use the optional incognito exception, turn on **Allow locks in incognito** in Settings and enable **Allow in Incognito** for 9 Bookmark in Chrome's extension details.
